Applications tierces (OAuth)
============================

Churros fourni un service de provider OAuth.

Des applications tierces peuvent utiliser l'API de Churros au nom d'une personne, si celle-ci autorise l'application.

Valeurs à utiliser
------------------

- **URL d'autorisation** 
  ```
  https://churros.inpt.fr/authorize
  ```
- **URL d'échange du code contre un token** 
  ```
  https://churros.inpt.fr/token
  ```

Plus de détails
---------------

Voir le guide sur [le wiki d'INP-net](https://wiki.inpt.fr/inp-net/public/oauth-churros) pour un guide sur l'intégration OAuth2.
